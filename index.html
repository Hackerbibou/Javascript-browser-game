Cardi G: A card guessing game

Instructions

Welcome to Cardi G.

  • First, you must memorize each card's position before clicking the start button.
  • Click the start button once you memorize the cards.
  • After clicking the start button, the timer will start counting down.
  • Now that the game has started, you must choose two matching cards.
  • When you get two matching cards right, the cards stay at the flipped position. Otherwise they flip back.
  • You only win when you get all the matching sets before timer runs out.
  • Goodluck!
